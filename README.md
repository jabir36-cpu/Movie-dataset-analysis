#  MovieLens Ratings Analysis
 Overview

This project analyzes the MovieLens ml-latest-small dataset using Python.
The goal is to explore movie ratings, understand user behavior, and identify patterns in movie genres and popularity.

This is an improved beginner-level data analysis project demonstrating practical use of data analytics tools.

Dataset Information


 ## The dataset contains:

- 100,000+ ratings

- 610 users

- 9,700+ movies

## Files used:

- movies.csv → movieId, title, genres

- ratings.csv → userId, movieId, rating, timestamp

- The datasets were merged using movieId for analysis.

## Technologies Used

- Python

- NumPy

- Pandas

- Matplotlib

- Seaborn

- Jupyter Notebook

## Analysis Performed

-  Data cleaning and preprocessing
- Merging multiple datasets
- Distribution of movie ratings
-  most rated movies
-  Average rating per movie
-  enre-based analysis
-  User rating behavior over time
-  Data visualization using graphs and charts

 ## Key Insights

- Some movies receive a high number of ratings but not necessarily high average ratings.

- Certain genres like Drama and Comedy are more frequently rated.

- User rating activity varies over time.

Popularity and quality (average rating) are different metrics.
